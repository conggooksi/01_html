■ form 내부에 쓰여지는 태그들 name="id" : 서버가 사용할 파라미터 이름==자바변수명==DB필드명 value="kim" : 서버가 사용할 데이터(값) text : 텍스트 입력할 때 사용 password : 암호 입력할 때 사용 radio : 라디오 버튼(오직 하나만 선택할 수 있다) checkbox : 체크박스(여러개 선택 가능, 하나도 선택 하지 않을 수도 있다) file : 파일 업로드할 때 사용(첨부파일) select : 콤보박스, 리스트 submit : 서버로 전송하는 명령 button : 클라이언트 javaScript함수 호출 할 때 사용 reset : form안의 입력한 데이터를 모두 지울 때
